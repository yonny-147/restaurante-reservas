# Sistema de Reservas - Restaurante

API REST para la gestión de reservas de un restaurante, construida con NestJS y
TypeScript. Proyecto académico del curso de Ingeniería de Software.

## Stack tecnológico

- NestJS + TypeScript
- Swagger / OpenAPI
- Jest (pruebas)

## Instalación

```bash
npm install
npm run start:dev
```

La API queda disponible en http://localhost:3000
